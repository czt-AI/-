# 《红楼水浒谍影江湖》项目说明

本项目旨在创作一部融合了《红楼梦》和《水浒传》元素的江湖谍战小说，讲述林黛玉和鲁智深在江湖中展开一场惊心动魄的谍战故事。

## 项目结构

- `source/`: 存放小说的原始文本文件。
- `notes/`: 存放创作笔记、大纲、人物设定等辅助文档。
- `assets/`: 存放小说相关的资源文件，如插图、地图等。

## 开发环境要求

- 操作系统：Windows/Linux/MacOS
- 编辑器：支持Markdown格式的文本编辑器，如Visual Studio Code、Sublime Text等。
